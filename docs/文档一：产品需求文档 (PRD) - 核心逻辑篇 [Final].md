### 📂 文档一：产品需求文档 (PRD) - 核心逻辑篇 [Final]

> **项目名称**：翻翻单词 (FlipWord)
> **Slogan**：Flip cards, Spark minds.
> **核心隐喻**：每一次“翻转”，都是一次思维的闪光。

#### 1. 核心玩法循环 (Core Loop)

1. **行动 (Action)**：在首页完成单词复习。核心动作是 **“Flip (翻转)”** 卡片。
2. **奖励 (Reward)**：复习每满 10 个词（或完成每日任务） -> 获得 1 个卡包。
3. **消耗 (Consume)**：在卡册页/详情页“撕开”卡包。
4. **积累 (Collection)**：
* **新卡**：解锁图鉴，点亮卡片。
* **重复卡**：自动分解为 **“星尘 (Stardust)”**。


5. **成长 (Progression)**：
* **熟练度提升**：SM-2 复习让单词从“生疏”变“掌握”。
* **星级提升**：熟练度达标 -> 卡片升星 -> 解锁新词形 (如 run -> running)。



#### 2. 数值与经济系统

* **❤️ 体力/卡包限制**：
* 免费用户：每日上限 5 包（登录送3 + 复习得2）。
* 订阅用户：每日上限 7-8 包（加速收集）。


* **💎 星尘 (Stardust)**：
* **定义**：卡牌分解后的魔法粉末。
* **来源**：重复卡分解 (普通5 / 稀有15 / 史诗50 / 传说200)、成就奖励。
* **用途**：**仅用于装饰**（兑换精美卡背、特效、头像框），**绝不**影响数值强度。


* **🃏 卡牌稀有度 (Rarity)**：
* **分布**：普通(60%)、稀有(25%)、史诗(12%)、传说(3%)。
* **机制**：稀有度绑定单词本身，不随机。



#### 3. 关键功能模块

| 模块 | 功能点 | 逻辑细节 |
| --- | --- | --- |
| **首页 (Home)** | **Hero 卡片** | 巨大的“今日复习”入口，带有轻微的呼吸动效，暗示用户去“翻转”它。 |
|  | **卡包槽** | 3D 卡包堆叠。若有可开卡包，显示“NEW”标记。 |
| **复习 (Review)** | **Flip 交互** | 屏幕中央显示单词卡 -> 点击卡片触发 **Flip 动画** -> 背面显示释义 -> 底部出现 SM-2 评分按钮。 |
| **开包 (Gacha)** | **抽卡逻辑** | 每次开包 5 张卡。**智能防脸黑**：优先出当前主题未收集的卡，但保留重复率以产出星尘。 |
| **卡册 (Binder)** | **词形系统** | 卡片初始 1 星。复习达到“掌握”状态后，解锁 2 星(复数/过去式)。 |
